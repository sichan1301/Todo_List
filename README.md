# 😍 Sichan's TodoList

## 😎 개요

- 오늘, 이번주, 이번달, 올해의 할일을 단위별로 기록해주는 계획표
- 할일 추가, 수정, 삭제, 완료 표시 기능
- 카테고리별로 localstorage 사용하여 나의 할일 저장
